# ScanFlow – Document Scanner Website

A professional website for the ScanFlow Document Scanner app, built with Markdown.

## Pages

| File | Page | Description |
|------|------|-------------|
| `index.md` | Home | Landing page with app overview and feature highlights |
| `features.md` | Features | Detailed breakdown of all 8 app features |
| `privacy-policy.md` | Privacy Policy | Full privacy policy for the app |
| `contact.md` | Contact | Contact information and support details |

## Features Covered

1. 📄 Document Scanner
2. ✏️ PDF Editor
3. ✍️ PDF Sign
4. 🖍️ Highlight & Annotate
5. 🔍 PDF to OCR
6. 📷 Image to OCR
7. 📱 QR Code Scanner
8. 🔗 QR Code Generator

## Deployment

These Markdown files can be used with any static site generator:

- **GitHub Pages** (Jekyll) — push to a repo and enable Pages
- **Hugo** — convert frontmatter as needed
- **MkDocs** — drop into `docs/` folder
- **Any Markdown renderer** — works as standalone content
